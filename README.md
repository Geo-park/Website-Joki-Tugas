# TugasIn Aja

Selamat datang di proyek landing page TugasIn Aja — situs sederhana untuk layanan joki tugas, presentasi, makalah, website, dan bantuan akademik.

## ✨ Fitur Utama
- Tampilan landing page yang rapi dan modern
- Navigasi antar halaman: Beranda, Layanan, Profil, Hubungi
- Form kontak yang terhubung ke WhatsApp
- Animasi ringan untuk membuat halaman lebih hidup
- Logo brand yang konsisten menggunakan simbol TJ

## 📁 Struktur File
- `index.html` — halaman utama
- `layanan.html` — daftar layanan dan harga
- `profil.html` — profil dan FAQ
- `hubungi.html` — form kontak
- `styles.css` — semua styling dan animasi
- `script.js` — menu mobile, FAQ, dan form kontak

## 🚀 Cara Menjalankan
1. Buka folder project ini.
2. Jalankan file `index.html` di browser.
3. Atau gunakan live server jika Anda ingin tampilan lebih nyaman saat mengedit.

## 💡 Catatan
Proyek ini dibuat dengan HTML, CSS, dan JavaScript sederhana, cocok untuk dipelajari atau dikembangkan lebih lanjut.
